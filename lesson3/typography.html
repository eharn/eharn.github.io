Headings Typography: Raleway, Helvetica, sans-serif

This font is ince and strong but still has a strong personality. This will give the site character without deterring the target audience.

Body Type: Montserrat, Arial, Helvetica, sans-serif

This font is similar to Raleway enough to make them have unity on the page, but they don't conflic both being sans serifs.

Footer Type: Montserrat, Arial, Helvetica, sans-serif

Reflecting the heading type will create more unity and consistency in the site.
